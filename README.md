Fitness-tracker

  個人開発の筋トレ記録アプリ。ワークアウトの記録・進捗の可視化・
  PR（自己ベスト）管理などを1ファイルで実現。

  ## 機能
  - ウエイトトレーニングの記録（種目・重量・回数・セット数）    
  - 有酸素運動の記録
  - 体重・体脂肪率の記録
  - カレンダービューで振り返り
  - グラフで進捗可視化
  - PR（パーソナルレコード）の自動検出

  ## 技術スタック
  - HTML / CSS / JavaScript（バニラ）
  - LocalStorageによるデータ永続化
  - 外部ライブラリなし・単一ファイル完結

  ## 使い方
  `fitness-tracker.html` をブラウザで開くだけで動作します。     

  ## 開発背景
  自分自身が筋トレをしていて「使いやすい記録アプリが欲しい」とい
  う自己課題から開発。
  トレーナー向けSaaS「PT Manager」の開発経験につながっています。
